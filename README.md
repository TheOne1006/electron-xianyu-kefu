# 🤖 electron-xianyu-kefu

基于 Electron 的闲鱼客服自动化工具，通过 AI 助手自动处理客户咨询，支持智能意图分类、自动回复、商品采集和对话管理。

> 🛠️ 本项目面向普通用户。如果你是开发者，想了解技术细节、参与开发或自行构建，请查看 [开发者文档](./DEV_README.md)。

## 1. ✨ 项目特点

1. **🌐 内嵌真实浏览器**：应用内直接加载闲鱼网页版，只需扫码登录一次，无需手动抓取 Cookie 或折腾复杂配置。
2. **📊 可视化管理**：提示词、商品库、文档知识库、对话记录全部通过图形界面管理，零门槛上手。
3. **🧩 组件化提示词**：知识库和提示词可自由组合，5 个专用 Agent 各司其职，覆盖议价、咨询、技术问答等场景。
4. **🛡️ 安全过滤**：内置关键词拦截机制，支持设置人工接管关键词，防止 AI 回复不当内容。

## 2. 🔄 自动回复流程

```md
        买家发来消息
          │
          ▼
    提取聊天数据（DOM 抓取）
          │
          ▼

记录消息 → 关联商品信息
│
▼
┌─ 意图分类 ─────────────────────────┐
│ 先看关键词： │
│ "便宜""砍价" → 价格类 │
│ "参数""型号" → 技术类 │
│ 没命中？交给 AI 判断： │
│ 议价 / 咨询 / 下单 / 问候 / 其他 │
└──────────┬─────────────────────────┘
│
▼
Agent 结合「提示词 + 商品信息 + 文档库 + 对话历史」
调用 LLM 生成回复
│
▼
安全过滤 → 入队待发送
│
▼
自动发送给买家
```

## 3. 🤖 Prompt 组成过程

每次 AI 回复前，系统会自动拼出一段完整的提示词发给大模型：

```md
┌─────────────────────────────────────────────────┐
│ 发给 AI 的完整 Prompt │
│ │
│ ┌───────────────────────────────────────────┐ │
│ │ ① Agent 提示词 │ │
│ │ 你在配置页面写的那段话， │ │
│ │ 比如 "你是一个专业的客服，语气要友好…" │ │
│ └───────────────────────────────────────────┘ │
│ ┌───────────────────────────────────────────┐ │
│ │ ② 商品信息（自动注入） │ │
│ │ 商品名称 / 价格 / 描述 │ │
│ └───────────────────────────────────────────┘ │
│ ┌───────────────────────────────────────────┐ │
│ │ ③ 文档知识库（可选，自动注入） │ │
│ │ 商品关联了哪些文档，内容就一起带上， │ │
│ │ 相当于给 AI 一份"参考手册" │ │
│ └───────────────────────────────────────────┘ │
│ │
│ + 对话历史（买家和卖家之前的聊天记录） │
└─────────────────────────────────────────────────┘
```

## 4. 🚀 快速开始

1. **下载安装**：到 [发布页面](https://github.com/TheOne/electron-xianyu-kefu/releases) 下载对应系统的安装包，双击安装即可。
2. **扫码登录**：打开应用后，点击「启动浏览器」，用微信扫码登录闲鱼账号。
3. **配置 AI**：在「配置」页面填入你的 AI 模型信息——包括模型名称（如 gpt-4o）、API 地址和 API 密钥。
4. **添加商品**：在「产品管理」页面添加你售卖的商品信息，AI 会参考这些信息生成更精准的回复。
5. **编辑提示词**：在「Agent 配置」页面调整 5 个 Agent 的提示词，让 AI 的回复风格更符合你的店铺调性。
6. **配置知识库文档**：在「文档管理」页面创建知识库文档，比如你可以写一份商品的常见问题解答，AI 回复时会自动引用这些文档内容。
7. **开始使用**：配置完成后，系统会自动检测并处理客户消息，你只需偶尔查看对话记录即可。

## 6. 📋 个性化设置

### Agent 提示词

应用内置 5 个专用 Agent，每个 Agent 负责不同场景：

- **system** — 系统级 Agent，定义整体行为规范
- **classify** — 意图分类 Agent，判断用户消息类型
- **default** — 默认回复 Agent，处理通用咨询
- **price** — 议价 Agent，处理价格相关对话
- **tech** — 技术 Agent，处理产品技术问题

在「Agent 配置」页面可以编辑每个 Agent 的提示词和推理参数，让 AI 回复更贴合你的需求。

### 安全过滤

在「配置」页面的「安全过滤」区域可以设置拦截关键词和替换字符。还可以设置「接管关键词」——当用户消息包含该关键词时，系统会暂停自动回复，等你手动处理。

### 文档知识库

在「文档管理」页面可以创建知识库文档。比如你可以写一份商品的常见问题解答，AI 回复时会自动引用这些文档内容。

## 7. ⚠️ 注意事项

1. **🔐 浏览器登录**：应用内嵌了真实的闲鱼网页，首次使用需要扫码登录，登录状态会自动保持。
2. **💰 API 费用**：每次自动回复都会调用 AI 模型，请注意服务商的计费规则，避免意外消耗。
3. **📜 合规使用**：本工具仅供学习和个人使用，请遵守闲鱼平台的使用条款。
4. **🔄 商品采集**：浏览闲鱼商品页面时，点击页面上的「📦 收集产品」浮动按钮，即可自动提取商品信息保存到产品列表。
